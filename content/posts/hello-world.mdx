## A quick code example

Here's the first JavaScript function I ever wrote:

```js
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("world")); // Hello, world!
```

And the TypeScript version once I levelled up:

```ts
const greet = (name: string): string => `Hello, ${name}!`;

console.log(greet("world"));
```
